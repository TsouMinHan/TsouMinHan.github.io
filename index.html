github pages + Hexo架設個人靜態部落格

發表於 2020-04-16

在看了些文章之後決定寫個部落格來玩玩看，文中提到可以透過文章讓別人了解自己會哪些技術或喜歡哪些事物，可以將部落格當作自己的作品集，並透過輸出的方式加深自己的印象。
有很多個平台可以選擇，參考許多文章後最後我是選擇github page+ Hexo，畢竟免費而且方便管理和佈署，但是缺點是github是開源的所以大家都有你的檔案，但是不要放些重要的東西都沒甚麼影響啦。
