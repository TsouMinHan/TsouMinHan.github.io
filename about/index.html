作品

  1. 個人助理（https://github.com/TsouMinHan/KiMoAssistant）

    在設定的時間內執行任務，透過爬蟲取得資料（書籍、文章、新聞等），並將其資訊儲存於資料庫中，提供DiscordBot與個人網站使用，使用 爬蟲 + sqlite 開發，約在大四時開發，在開發中使用過heroku部署程式，用過PHPMyadmin、PostgreSQL和Firebase儲存資料，最後選擇使用sqlite，直到現在依然持續維護與新增功能，目前在樹莓派上運行中。

  2. 個人網站（https://github.com/TsouMinHan/KiWeb）

    用於顯示爬取到的新聞文章、書籍查詢以及資料庫管理，使用 Flask + Jinja2 開發，約在大四時開發，直到現在依然持續維護與新增功能，過去將網站部署在GAE上，目前在樹莓派上運行中。

  3. 推播通知

    最開始是使用Line Notify推播，原本是個人助理程式中的功能之一，直到畢業後才改用DiscordBot推播，獨立成為一個專案。

    使用DiscordBot將個人助理所執行的結果推播到Discord中，另可透過指令顯示各地區近三日的氣象狀況，使用 python + DiscordBot + 爬蟲 + sqlite 開發，在碩士畢業後開發，直到現在依然持續維護與新增功能，目前在樹莓派上運行中。

  4. 專案執行管理平台（https://github.com/TsouMinHan/ProjectCenter）

    為了節省開啟虛擬環境執行程式或撰寫bat檔案的步驟，只要打開這個程式就可以直接執行目錄中所有的專案程式了，使用 eel + JavaScript 開發，大約花了1天完成。

  5. 專案產生器（https://github.com/TsouMinHan/PorjectMaker）

    為了省略每次新建專案要手動產生固定的檔案、虛擬環境與git的步驟而產生的，使用 python 開發，大約花了1天完成以及多次的修正。

  6. 簡易QRCode 產生器（https://github.com/TsouMinHan/QRCodeGenerator）

    在老師的建議下製作的，產生QRcode後將文字直接顯示在圖片下方，使用 eel + JavaScript 開發，大約花了1個下午完成。

  7. 人臉辨識點名推播系統（https://github.com/TsouMinHan/Check-inSystem）

    這個作品是論文的前身，使用手機拍攝畫面即時傳輸至後端，在後端伺服器辨識後推播通知，使用 LineNotify + Dlib + Flask + webrtc 開發，大約花了2週完成以及多次的修改。

  8. 音樂分配調整器（https://github.com/TsouMinHan/MusicAmplifier）

    一次對數個特定格式的音樂檔案進行分貝調整，使用 eel + Javascript 開發，大約花了2天完成。

  9. 目標追蹤（https://github.com/TsouMinHan/TargetHours）

    當時想要追蹤自己在每個項目的執行進度而製作的，但最後卻沒有使用，就當作練習吧，使用 PyQt5 開發，大約花了2個晚上完成。

  10. 沒屋頂的網站圖片下載（https://github.com/TsouMinHan/NoRoof）

    在同學的請求下撰寫的，可以將賣家所有分類下的商品圖片下載下來，但有個小bug尚未修改，只會爬分類下第一頁的商品，所以當品項更多的時候就只會爬取部份商品。

  11. 語音轉文字（https://github.com/TsouMinHan/ShowYourWords）

    可將語音轉換成文字給聽障生看，使用 PyQt5 + gtts 開發，大約花了1個晚上完成。

  12. 貪食蛇遊戲（https://github.com/TsouMinHan/-/tree/master/game/snake）

    在深碗課程中讓學生初步了解pygame的應用，使用 pygame 開發，大約花了2個晚上完成，還有很多地方需要改進的部份。
